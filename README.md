# n8n-Pick-a-daily-Facebook-comment-contest-winner-with-OpenAI-Airtable-Telegram
Run fair, hands-free Facebook giveaways every day with this n8n workflow automation. It scans post comments, excludes past winners from Airtable, uses OpenAI to filter genuine positive entries, picks a random winner, saves results and announces them on Telegram using this reliable n8n workflow template.
